# User Fetch (Vanilla JavaScript)

Aplicação web para consumo de API utilizando JavaScript puro, com foco em requisições assíncronas, tratamento de erros e manipulação dinâmica do DOM.

---

## Funcionalidades

* Requisição de dados de usuário via API
* Exibição dinâmica dos dados na interface
* Tratamento de erros de requisição
* Controle de estado da interface (erro e estado vazio)

---

## Tecnologias

* JavaScript (ES Modules)
* HTML
* CSS

---

## Destaques Técnicos

* Uso de `async/await` para requisições assíncronas
* Tratamento de erros com `try/catch`
* Validação de resposta da API
* Manipulação direta do DOM
* Separação de responsabilidades (requisição e renderização)

---

## Execução

```bash
git clone <repo-url>
cd <nome-do-projeto>
```

Abra o arquivo `index.html` no navegador.

---

## Objetivo

Demonstrar conhecimento em consumo de APIs, tratamento de dados assíncronos e atualização de interface utilizando JavaScript moderno.
